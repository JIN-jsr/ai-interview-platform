# 演示视频脚本

建议视频控制在 6-8 分钟。

## 1. 项目启动

展示命令：

```bash
.venv\Scripts\activate
streamlit run app.py
```

说明系统支持本地运行。

## 2. 简历输入与解析

展示上传或粘贴简历，点击“解析简历并生成画像”。

重点说明：

- 支持 TXT / PDF / DOCX
- 可生成结构化简历 JSON
- 可生成用户画像与面试重点

## 3. RAG 知识库

进入“RAG 知识库”页面。

展示：

- 知识库 80 条
- 覆盖多个计算机基础方向
- 搜索 `Python MySQL Redis 后端`
- 展示检索出的题目、参考答案和追问

## 4. 模拟面试

进入“模拟面试”页面。

展示：

- 自我介绍
- 项目经历深挖
- 用户回答中提到 Redis / MySQL
- 系统根据关键词追问
- RAG 基础知识题

## 5. 面试记录与分析

展示每轮回答记录：

- 问题类型
- 用户回答
- 技术关键词
- 覆盖点与缺失点
- 临时评分
- 改进建议

## 6. 评分报告

进入“评分报告”页面，点击“生成正式评分报告”。

展示：

- 总分
- 等级
- 五维度评分
- 评分依据
- 主要问题
- 提升建议
- 下载 JSON / Markdown

## 7. 项目结构

简要展示 GitHub 仓库结构：

- README
- requirements.txt
- data/knowledge_base.json
- src/
- docs/
- .env.example

## 8. 结尾

说明系统已完整打通：

简历输入 → 用户画像 → RAG 检索 → 连续面试 → 项目追问 → 回答分析 → 评分反馈
