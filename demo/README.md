# 演示数据说明

`demo/` 目录中的材料均为虚构内容，只用于系统测试、比赛演示和视频录制，不包含真实个人信息。

侧边栏“演示模式”会根据选择的示例简历创建一场新的演示面试，并加载对应岗位的回答模板，方便录屏时复制相近回答。

## 示例简历

| 文件 | 目标岗位 |
|---|---|
| `sample_resume_backend.txt` | 后端开发 |
| `sample_resume_frontend.txt` | 前端开发 |
| `sample_resume_ai_app.txt` | AI 应用开发 |
| `sample_resume_data_analysis.txt` | 数据分析 |
| `sample_resume_testing.txt` | 软件测试 |

## 使用建议

1. 启动应用。
2. 在侧边栏打开“演示模式”。
3. 选择目标岗位示例简历。
4. 点击“一键加载示例简历”。
5. 解析简历并开始模拟面试。
6. 根据当前问题选择对应回答模板中的内容进行演示。

提交 GitHub 前，请不要把真实简历、真实面试记录、真实评分报告或真实 API Key 放入本目录。
