# 演示数据说明

`demo/` 目录中的材料均为虚构内容，只用于系统测试、比赛演示和视频录制。

这些文件不包含真实个人信息，不应替代真实求职建议，也不应与真实候选人数据混用。录制演示视频时，可以在侧边栏“演示模式”中加载示例简历，再根据问题选择相近的回答模板复制到面试输入框。

## 示例简历

- `sample_resume_backend.txt`：后端开发示例
- `sample_resume_frontend.txt`：前端开发示例
- `sample_resume_ai_app.txt`：AI 应用开发示例
- `sample_resume_data_analysis.txt`：数据分析示例
- `sample_resume_testing.txt`：软件测试示例

## 示例回答

- `demo_answers.md`：通用回答模板
- `sample_answers_backend.md`：后端开发回答模板
- `sample_answers_frontend.md`：前端开发回答模板
- `sample_answers_ai_app.md`：AI 应用开发回答模板
- `sample_answers_data_analysis.md`：数据分析回答模板
- `sample_answers_testing.md`：软件测试回答模板

提交 GitHub 前，请不要把真实简历、真实面试记录、真实评分报告或真实 API Key 放入本目录。
