# 数据分析演示回答模板

## 自我介绍
我主要准备数据分析岗位，熟悉 SQL、Pandas、Excel 和可视化报表。项目中做过电商用户行为分析，负责数据清洗、指标口径梳理、转化漏斗和留存分析。

## 项目介绍
电商用户行为分析平台主要分析浏览、加购、下单和复购链路。我负责清洗日志数据、统一指标口径，并用 SQL 聚合和窗口函数计算转化率、留存率、客单价等指标，最后用报表展示不同用户分层的表现。

## 数据清洗
我会先检查缺失值、重复值、异常值和字段口径是否一致。对关键指标字段会保留处理规则，例如订单金额异常、测试账号过滤和时间范围统一，保证后续分析结果可信。

## 指标体系
我会围绕业务目标拆解指标，例如拉新看新增用户数，转化看浏览到下单转化率，留存看次日和七日留存，收入看客单价和 GMV。每个指标都要明确口径、时间范围和数据来源。
