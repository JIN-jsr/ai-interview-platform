# 前端开发演示回答模板

以下内容为虚构演示材料，仅用于系统测试和视频录制。

## 自我介绍

我主要准备前端开发岗位，熟悉 HTML、CSS、JavaScript、TypeScript，也使用过 Vue 和 React。项目中我做过活动管理平台和智能简历面试页面，比较关注组件拆分、接口联调、页面状态和响应式布局。

## 组件化

组件化的核心是把可复用的 UI 和交互逻辑拆出来，例如表单、表格、弹窗和上传组件。拆分时我会注意 props 设计、事件回调、状态归属和样式隔离，避免组件过度耦合。这样后续新增页面时可以复用已有组件，减少重复代码。

## 状态管理

我会根据状态影响范围选择管理方式。只在当前组件使用的状态可以放本地，跨多个页面或多个组件共享的用户信息、筛选条件和权限信息可以放到统一状态管理里。同时要避免把所有状态都全局化，否则会增加维护成本。

## 接口联调

接口联调时我会先确认请求方法、参数、鉴权方式和响应结构，然后用 Axios 封装统一请求。对加载中、空数据、错误提示和重试场景都要处理。遇到问题时会结合浏览器 Network、后端日志和接口文档定位。

## 响应式布局

响应式布局需要根据屏幕宽度调整栅格、字体、按钮和内容密度。常见做法包括 flex、grid、媒体查询和合理的 min/max 宽度。我会优先保证核心信息在移动端不溢出、不遮挡，并保持操作按钮容易点击。
